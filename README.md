# Expense Tracker 💰

Expense Tracker is a Flutter application designed to help users manage daily income and expenses in a simple and organized way.

## Features ✨

* Add income and expense records
* Track financial transactions
* Simple and clean user interface
* Easy navigation and management
* Built with Flutter for cross-platform support

## Technologies Used 🛠️

* Flutter
* Dart

## Project Structure 📂

```text
lib/
assets/
```

## Installation 🚀

Clone the repository:

```bash
git clone https://github.com/Mostafa-Adnan/ExpenseTracker.git
```

Move to the project folder:

```bash
cd ExpenseTracker
```

Install dependencies:

```bash
flutter pub get
```

Run the application:

```bash
flutter run
```
